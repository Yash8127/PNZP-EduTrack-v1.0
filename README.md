🎓 EduTrack v1.0

EduTrack is a School Management System designed to simplify and digitalize daily academic and administrative operations. It helps manage students, teachers, classes, exams, and fees efficiently.

🚀 **Features**
👨‍🎓 Student Management
👩‍🏫 Teacher Management
🏫 Class & Department Management
📝 Exam & Marks Tracking
💰 Fee Management System
🔐 User Authentication (Login/Register)
📊 Dashboard Overview
📁 Database Integration
🛠️ Tech Stack

**Frontend:**

HTML5
CSS3
Bootstrap 5
JSP (Java Server Pages)

**Backend:**

Java (Servlets & JDBC)

**Database:**

MySQL (tellabadu_school_db)

**Tools:**

Eclipse IDE
Apache Tomcat Server
Git & GitHub
